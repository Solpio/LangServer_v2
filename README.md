# LangServer_v2
## RESTful API 
Node.js + Typescript + Express + MongoDB

### API:
#### Auth
```
POST /auth/registration
POST /auth/login
GET /auth/me
```

#### Words
```
GET /words
```
#### Book
```
GET /book/sections
```
#### UserWords
```
POST /user/words
GET /user/words
GET /user/words/:id
PUT /user/words/:id/:status
```
